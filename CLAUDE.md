# "The Blueprint" Design Language

## Core Elements

**Colors:**

- Background: Pure White `#FFFFFF` or light gray grid
- Primary: Pure Black `#000000` (text, borders)
- Accent: Safety Yellow `#FACC15` (Tailwind `yellow-400`) - use sparingly
- Error/Strike: Red `#EF4444` (Tailwind `red-500`)
- Success: Green `#22C55E` (Tailwind `green-500`)
- Info: Blue `#3B82F6` (Tailwind `blue-500`)

**Typography:**

- Font: JetBrains Mono Bold (ONLY)
- Hierarchy: Massive contrast (8xl-9xl vs sm-base)

**Styling:**

- Shadows: "Hard Drop" `4px 4px 0px 0px #000000` (no blur)
- Borders: 3-4px thick (never 1px)
- Corners: 0px (sharp edges only)
- Spacing: Exaggerated whitespace

**Backgrounds:**

- Dot-grid pattern (engineering paper)
- Diagonal stripes (subtle)
- Pure white or light gray

**Layout Principles:**

- High contrast, brutalist aesthetic
- Technical/engineering vibe
- Scannable at thumbnail size
- Yellow highlights ONLY the most important word/number
- Text hierarchy: `[ LABEL ]` (small, gray) → MAIN TEXT (massive, black) → HIGHLIGHT (yellow)
